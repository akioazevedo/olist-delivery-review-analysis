# Olist Final Project

This project analyzes how delivery time, order value, payment installments,
and customer location affect customer review scores using the Olist dataset.

## How to run
1. Download the Olist dataset from Kaggle
2. Place CSV files inside a `data/` folder
3. Run `Final Project_Rcode_AkioMaebayashi.R`

## Output
Charts are saved in the `latex/` folder.
